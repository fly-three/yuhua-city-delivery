# 长沙雨花城配分单台

面向“2B 仓、多货主、城配承运商无系统且无面单”的末端配送场景。

## 核心设计

- 调度侧维护运单池，字段包含仓库、货主、收件人、收件地址、温层、重量、物品类型、件数，并支持自定义扩展字段。
- 按片区、温层、载重、时效、货主策略自动分单，生成司机路线、序号和交接清单。
- 不强依赖司机端：可打印 A4 交接单和包裹贴，也可复制司机 H5 链接让司机在手机浏览器里电话、导航、签收、报异常。
- 额外扩展：冷链/常温约束、多货主合并或拆分、CSV 交接导出、可携带数据的无后端司机链接、路线示意图。

## 本地运行

```bash
npm run dev
```

访问 http://localhost:4173

## Vercel

项目已包含 `vercel.json` 和 `npm run build`，Vercel 构建后输出目录为 `dist`。
